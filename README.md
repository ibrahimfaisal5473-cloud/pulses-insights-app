# Pulses Insights App

A visitor insights dashboard — visits, happiness/sentiment, zone analytics, and visitor journeys — built as part of the Phase 2 application development assignment.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- React + TypeScript
- Tailwind CSS

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

```
src/
  app/            # App Router pages, layouts, and global styles
```
